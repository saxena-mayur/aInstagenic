# aInstagenic

On Instagram, there’s no room for “good enough” shots. Time to level up using AI.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install packages.

```bash
pip install requirements.txt
```

### Data Curation

```bash
cd data_curation
sh download.sh
```

### Run the App

```bash
streamlit run app.py
```
